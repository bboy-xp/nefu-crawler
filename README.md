# 东北林业大学学生信息爬取程序

**项目简介：&#x1F449;**


&#x1F625;鉴于东北林业大学考场信息查询网站经常性的会非常缓慢，时不时的还会报错
&#x1F4A1;为了同学们能够稳定快速的查询课表、成绩、考场、补考信息
&#x1F4AA;我基于Node.js、axios、cheerio、MongoDB编写了爬虫程序


> &#x26A0; Tips：请大家善意测试，最好是在夜深人静的时候....避免占用教务处服务器大量资源影响教务性能


## 要求

  + Node.js v8.9.0
  + MongoDB 数据库已启动，运行端口27017

## 安装

```bash
  # 安装依赖
  $ npm install
```
